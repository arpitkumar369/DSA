<h1> Pattern 2 </h1>

<h2> Easy </h2>

1. You are given a number n.
2. You've to create a pattern of * and separated by tab as shown in output format.

<h3> Constraints </h3>

1 

<h3> Format </h3>

<h4> Input </h4>

A number n

<h4> Output </h4>
<h6>
  * * * * *<br>
  * * * * <br>
  * * * <br>
  * * <br>
  *
</h6>

<h3> Example </h3>

<h4> Sample Input </h4>

5

<h4> Sample Output </h4>

<h6>*	*	*	*	*	<br>
*	*	*	*	<br>
*	*	*	<br>
*	*	<br>
*</h6>
